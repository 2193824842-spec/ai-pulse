什么是 Cursor？

Cursor 是一款基于 VS Code 构建的 AI 代码编辑器，深度融合了 AI 与编码工作流。它支持多文件编辑、代码库感知聊天以及用于处理复杂任务的自主代理模式。Cursor 采用最前沿的模型（GPT-4、Claude），已成为 AI 辅助开发的首选编辑器。

核心功能

AI 驱动的代码编辑器（VS Code 衍生版）
Tab：智能多行自动补全
Cmd+K：行内代码生成和编辑
Chat：代码库感知的 AI 助手
Composer：多文件 AI 编辑代理
Agent 模式用于自主任务执行
代码库索引以实现上下文感知建议
支持 GPT-4、Claude 和自定义模型
集成终端 AI 辅助
Git 集成 AI 提交消息
隐私模式：代码永不存储在服务器上
兼容所有 VS Code 扩展

优缺点

👍 优点

  • 最佳多文件 AI 编辑 — Composer 可以重构整个代码库
  • 代码库索引为 AI 提供项目的完整上下文
  • 熟悉的 VS Code 界面 — 现有用户零学习曲线
  • Agent 模式自主处理复杂的多步骤编码任务
  • 支持多个前沿模型（GPT-4、Claude、Gemini）

👎 缺点

  • 正式使用需要 Pro 计划（$20/月）
  • 在大型代码库索引时可能较慢
  • Agent 模式可能做出意外更改——需要仔细审查
  • Business 计划价格昂贵，$40/用户/月
  • 依赖第三方 AI 模型（OpenAI、Anthropic）
