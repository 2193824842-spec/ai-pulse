什么是 DALL-E 3？

DALL-E 3 是 OpenAI 最先进的图像生成模型，已直接集成到 ChatGPT 中。它在准确理解和执行复杂详细的提示词方面表现出众，相比之前的版本有了重大进步。可通过 ChatGPT Plus 和 OpenAI API 获取使用。

核心功能

高保真文本到图像生成
精准提示词执行（相比 DALL-E 2 有重大改进）
直接集成到 ChatGPT 中，支持对话式图像创建
多种宽高比（正方形、横向、纵向）
通过 API 进行图像修补和编辑
通过 ChatGPT 进行自然语言提示词优化
为开发者提供 API 访问
内置安全内容过滤
高分辨率输出（1024x1024、1792x1024、1024x1792）
通过 C2PA 元数据进行水印标记

定价方案

方案价格包含内容
ChatGPT Plus $20/mo DALL-E 3 via ChatGPT · Limited generations per day · Conversational image editing
API - Standard $0.040/image 1024x1024 resolution · Pay per image · No subscription needed
API - HD $0.080/image 1024x1024 HD quality · Higher detail and fidelity · Pay per image

优缺点

👍 优点

  • 任何图像模型中最强的提示词遵循能力——准确执行复杂指令
  • 无缝集成至 ChatGPT 中用于对话工作流
  • 无需单独订阅——包含在 ChatGPT Plus 中
  • 简洁易用的开发者友好型 API
  • 在各种风格中保持一致的质量

👎 缺点

  • 没有免费的图像生成层级
  • 相比 Midjourney 艺术性和风格化程度较低
  • 与 Stable Diffusion 相比编辑和修复功能有限
  • 缺少风格参考和角色一致性功能
  • 大批量生成时 API 价格累加成本较高
