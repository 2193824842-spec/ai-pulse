What is Cursor?

Cursor is an AI-powered code editor built on VS Code that deeply integrates AI into the coding workflow. It supports multi-file editing, codebase-aware chat, and autonomous agent mode for complex tasks. Cursor uses frontier models (GPT-4, Claude) and has become the preferred editor for AI-assisted development.

Pros & Cons

👍 Pros

  • Best multi-file AI editing — Composer can refactor entire codebases
  • Codebase indexing gives AI full context of your project
  • Familiar VS Code interface — zero learning curve for existing users
  • Agent mode handles complex, multi-step coding tasks autonomously
  • Supports multiple frontier models (GPT-4, Claude, Gemini)

👎 Cons

  • Pro plan required for serious use ($20/mo)
  • Can be slow on large codebases during indexing
  • Agent mode can make unexpected changes — requires careful review
  • Business plan is expensive at $40/user/mo
  • Dependent on third-party AI models (OpenAI, Anthropic)
