What is Perplexity?

Perplexity is an AI-powered search engine that combines real-time web search with large language models to provide direct, cited answers to questions. Unlike traditional search engines, it synthesizes information from multiple sources and presents a clear answer with references — making research faster and more reliable.

Core Features

Real-time web search with AI synthesis
Cited sources for every answer
Follow-up questions in conversational threads
Focus modes: Web, Academic, YouTube, Reddit, News
Perplexity Pages: create shareable research documents
File and image analysis
Collections for organizing research
Pro Search with deeper multi-step reasoning
API access for developers
Mobile apps for iOS and Android
Browser extension
Multiple AI models (GPT-4, Claude, Sonar)

Pros & Cons

👍 Pros

  • Best AI search tool — real-time answers with cited sources
  • Academic focus mode for research papers
  • Conversational follow-ups maintain context
  • Much faster than traditional research workflows
  • Generous free tier with unlimited basic searches

👎 Cons

  • Can occasionally hallucinate or misattribute sources
  • Pro searches limited to 600/day even on paid plan
  • Less capable than ChatGPT for creative or coding tasks
  • Sources can be low-quality websites
  • No image generation
